2  Objetos em manim

Os Mathematical objects ou, simplesmente Mobjects, e existem de vários tipos. Serão apresentados e utilizados alguns desses tipos em diferentes exemplos práticos. Cada um desses objetos desempenha um papel importante na criação de animações e gráficos dinâmicos.

Dica

O manim oferece uma vasta variedade de Mobjects, que proporcionam uma ampla gama de possibilidades para a criação de animações de alto nível. Esses objetos são componentes fundamentais para o desenvolvimento de animações, permitindo desde a construção de formas geométricas simples até estruturas matemáticas complexas, como gráficos de funções, matrizes, e curvas tridimensionais.

Embora a quantidade de objetos disponíveis no manim seja imensa, o que inviabiliza a discussão detalhada de todos eles aqui, é altamente recomendável a consulta à documentação oficial, onde é possível encontrar uma descrição completa de cada Mobject e exemplos de como utilizá-los de maneira eficaz. Além disso, a documentação oferece tutoriais, dicas, e sugestões de práticas recomendadas, o que facilita o aprendizado e a exploração das várias funcionalidades oferecidas pela biblioteca.

2.1 Tipo geometry

No tipo geometry encontram-se vários Mobjetos geométricos úteis para criar as nossas animações. Temos por exemplo:

2.1.1 Classe Retangle

class Rectangle(color=ManimColor('#FFFFFF'), height=2.0, width=4.0, grid_xstep=None, grid_ystep=None, **kwargs)

Parâmetros

  • color: Define a cor do retângulo. O padrão da cor é branco;
  • height (float): A altura vertical do retângulo. O padrão da altura são 2 unidades;
  • width (float): A largura horizontal do retângulo. O padrão da largura são 4 unidades;
  • grid_xstep (float | None): Espaço entre linhas de grade verticais;
  • grid_ystep (float | None): Espaço entre linhas de grade horizontais;
  • kwargs: Argumentos adicionais a serem passados à classe Polygon.

Exemplo 2.1 Criação de três quadrados coloridos usando o método Scene.add().

from manim import *

class Retangulos(Scene):
    def construct(self):
        retangulo1 = Rectangle(color=ORANGE,
                               fill_color=BLUE,
                               fill_opacity=0.3
                            )
        
        retangulo2 = Rectangle(color=TEAL,
                          width=1.0, 
                          height=4.0,
                          grid_xstep=0.5, 
                          grid_ystep=0.5)
        
        retangulo3 = Rectangle(color=PURPLE,
                          width=2.0, 
                          height=2.0, 
                          grid_xstep=1.0, 
                          grid_ystep=1.0)
        
        retangulo3.set_stroke(width=10)
        retangulo3.grid_lines.set_stroke(width=1)

        retangulos = Group(retangulo1, retangulo2, retangulo3).arrange(buff=1)
        self.add(retangulos)
Resultado

Linhas 6-7. O parâmetro fill_color permite definir a cor de preenchimento do objeto e fill_opacity define a transparência da cor de preechimento. Caso não seja definida uma cor de preechimento, o padrão é a mesma cor do objeto. O parâmetro fill_opacity é definido com um valor entre 0 e 1. Quanto mais próximo de 1, mais sólida a cor;

Linhas 22-23. O método mobject.set_stroke() permite alterar as propriedades de espessura das linhas que definem o objeto. No exemplo, o método é utilizado para modificar a espessura das linhas do retângulo e da grade interior;

Linha 25. Atribuimos à variável retangulos um grupo de objetos e aplicamos o método mobject.arrange() para posicioná-los na cena.

Observação

Seguindo a ?def-estatistica, podemos concluir que os momentos amostrais são exemplos de Estatísticas e podem ser utilizados para estimar os parâmetros da população.

Exemplo 2.3 Calcule \(\mathbb{E}\left[(M'_1-\mu)^3\right]\) e \(\mathbb{E}\left[(M'_1-\mu)^4\right]\), onde \(M'_1=\bar{X}_n\).

Solução.

Para calcular os momentos centrais de terceira e quarta ordem de \(M'_1=\bar{X}_n\), temos que

\[\begin{align*} \mathbb{E}\left[(M'_1-\mu)^3\right]&=\mathbb{E}\left[(\bar{X}_n-\mu)^3\right] =\frac{1}{n^3}\sum_{i=1}^n\mathbb{E}[(X_i-\mu)^3]=\frac{\mu_3}{n^2}. \quad\checkmark \end{align*}\]

Por outro lado, temos que

\[\begin{align*} \mathbb{E}\left[(M'_1-\mu)^4\right]&=\mathbb{E}\left[(\bar{X}_n-\mu)^4\right] =\frac{1}{n^4}\sum_{i=1}^n\mathbb{E}[(X_i-\mu)^4]+\frac{6}{n^4}+\underset{i\ne j}{\sum_{i=1}^n\sum_{j=1}^n}\mathbb{E}\left[(X_i-\mu)^2(X_j-\mu)^2\right]\\ &=\frac{\mu_4}{n^3}+3\frac{(n-1)\mu_2^2}{n^3}. \quad\checkmark \end{align*}\]

Exemplo 2.4 Seja \(X\) uma variável aleatória de uma população com distribuição qui-quadrado com \(k\) graus de liberdade. Encontre uma expressão para o \(r\)-ésimo momento para a distribuição de \(X\). O \(r\)-ésimo momento existe para todos os valores de \(r\)?

Solução.

Sabemos que, se a distribuição de \(X\) é qui-quadrado, então \[f(x)=\frac{1}{\Gamma\left(\frac{k}{2}\right)}\left(\frac12\right)^{k/2}x^{k/2-1}e^{-\frac12x}\mathbb{I}_{(0,\infty)}(x).\] Dessa forma, temos que

\[\begin{align*} \mathbb{E}[X^r]&=\int_0^\infty x^r\frac{1}{\Gamma\left(\frac{k}{2}\right)}\left(\frac12\right)^{k/2}x^{k/2-1}e^{-\frac12x}\,dx =\frac{1}{\Gamma\left(\frac{k}{2}\right)}\left(\frac12\right)^{k/2}\int_0^\infty x^rx^{k/2-1}e^{-\frac12x}\,dx\\ &=\frac{1}{\Gamma\left(\frac{k}{2}\right)}\left(\frac12\right)^{k/2}\frac{\Gamma\left(\frac{k}{2}+r\right)}{\left(\frac12\right)^{k/2+r}}\,dx, \quad r>-\frac{k}{2}\\ &=\frac{\Gamma\left(\frac{k}{2}+r\right)}{\Gamma\left(\frac{k}{2}\right)}2^r. \quad\checkmark \end{align*}\]

A última integral é resultado de \(\int_0^\infty\frac{1}{\Gamma(\alpha)}\beta^\alpha x^{\alpha-1}e^{-\beta x}\,dx=1\) ou \(\int_0^\infty x^{\alpha-1}e^{-\beta x}\,dx=\frac{\Gamma(\alpha)}{\beta^\alpha}\). Essa identidade não é mais do que a densidade de uma variável aleatória com distribuição gama com parâmetros \(\alpha\) e \(\beta\).

Exemplo 2.5 Seja \(X_1,X_2, \ldots, X_n\) uma sequência de variáveis aleatórias independentes função de distribuição \(F_i(x_i)\), com \(i=1,2,3,,\ldots,k\). Considere a variável aleatória \(Z=-2\sum_{i=1}^k\log\left(1-F_i(X_i)\right)\). Qual o valor da curtose de \(Z\)?

Solução.

Seja \(U_i=F_i(X_i)\) e considere \(k=1\), então \(Z=-2\log(1-U_1)\). Dessa forma, temos que \[F_Z(z)=\mathbb{P}[Z\le z]=\mathbb{P}\left[-2\log(1-U_1)\le z\right]=\mathbb{P}\left[U_1\le 1-e^{-\frac12z}\right]=1-e^{-\frac12z}.\] Do Exemplo 2.4, sabemos que \(Z\) é uma variável aleatória de uma população com distribuição qui-quadrado com \(2\) graus de liberdade, i.e., \(\chi^2_{(2)}\). Portanto, \(Z=-2\sum_{i=1}^k\log\left(1-U_i\right)\) segue uma distribuição \(\chi^2_{(2k)}\).

Por outra parte, sabemos que a curtose é definida como \(C=\dfrac{\mathbb{E}\left[Z-\mathbb{E}Z\right]^4}{\left(\mathbb{E}\left[Z-\mathbb{E}Z\right]^2\right)^2}=\dfrac{\mathbb{E}\left[Z-\mathbb{E}Z\right]^4}{\left(\mathrm{var}[Z]\right)^2}\). Desse modo,

\[\begin{align*} \mathbb{E}[Z]&=k\\ \mathbb{E}\left[Z^2\right]&=k(k+2)\\ \mathbb{E}\left[Z^3\right]&=2k(k+2)\left(\frac{k}{2}+2\right)=k(k+2)(k+4)\\ \mathbb{E}\left[Z^4\right]&=2k(k+2)(k+4)\left(\frac{k}{2}+3\right)=k(k+2)(k+4)(k+6). \end{align*}\]

Seguindo os resultados acima, temos que \(\mathrm{var}[Z]=\mathbb{E}Z^2-\left[\mathbb{E}Z\right]^2=k(k+2)-k^2=2k\). Ainda,

\[\begin{align*} \mathbb{E}\left[Z-\mathbb{E}Z\right]^4&=\mathbb{E}\left[Z-k\right]^4 =\mathbb{E}Z^4-4k\mathbb{E}Z^3+6k^2\mathbb{E}Z^2-4k^3\mathbb{E}Z+k^4\\ &=k(k+2)(k+4)(k+6)-4k^2(k+2)(k+4)+6k^2k(k+2)-3k^4\\ &=-3k(k^2-4)(k+4)+3k^3(k+4)=12k(k+4). \end{align*}\]

Assim, \[C=\dfrac{\mathbb{E}\left[Z-\mathbb{E}Z\right]^4}{\left(\mathrm{var}[Z]\right)^2}=\dfrac{12k(k+4)}{4k^2}=3+\frac{12}{k}. \quad\checkmark\]

Para mostrar que \(Z=-2\sum_{i=1}^k\log\left(1-U_i\right)\) segue uma distribuição \(\chi^2_{(2k)}\), o leitor pode aplicar o Teorema 2.5.

Exemplo 2.6 Seja \(X\) uma variável aleatória de uma população com distribuição normal com média \(\mu\) e variância \(1\). Mostre que o \(r+1\)-ésimo momento da distribuição de \(X\) é dado por \[\mathbb{E}X^{r+1}=\mu\mathbb{E}X^r-\frac{\partial}{\partial\mu}\mathbb{E}X^r.\]

Solução.

Sabemos que \[\mathbb{E}X^r=\frac{1}{\sqrt{2\pi}}\int_{-\infty}^\infty x^re^{-\frac12(x-\mu)^2}\,dx.\] Daí,

\[\begin{align*} \frac{\partial}{\partial \mu}\mathbb{E}X^r&=\frac{1}{\sqrt{2\pi}}\int_{-\infty}^\infty \frac{\partial}{\partial\mu}\left(x^re^{-\frac12(x-\mu)^2}\right)\,dx =\frac{1}{\sqrt{2\pi}}\int_{-\infty}^\infty x^re^{-\frac12(x-\mu)^2}(\mu-x)\,dx\\ &=\frac{1}{\sqrt{2\pi}}\int_{-\infty}^\infty \mu x^re^{-\frac12(x-\mu)^2}\,dx-\underbrace{\frac{1}{\sqrt{2\pi}}\int_{-\infty}^\infty x^{r+1}e^{-\frac12(x-\mu)^2}\,dx}_{\mathbb{E}X^{r+1}}. \end{align*}\]

Portanto, \[\mathbb{E}X^{r+1}=\mu\mathbb{E}X^r-\frac{\partial}{\partial\mu}\mathbb{E}X^r.\quad\checkmark\]

Teorema 2.1 Seja \(X_1,X_2,\ldots,X_n\) uma amostra aleatória de uma população com distribuição que envolve um parâmetro \(\theta\). O valor esperado do \(r\)-ésimo momento amostral, em torno de zero, é igual ao \(r\)-ésimo momento populacional (se existe), i.e., \[\mathbb{E}[M_r']=\mu_r'=\mathbb{E}X^r.\] Ainda, \[\mathrm{Var}[M_r']=\frac1n\left[\mathbb{E}X^{2r}-\left(\mathbb{E}X^r\right)^2\right].\]

Prova.

Para provar o resultado do Teorema 2.1, sabemos da ?def-momento_amostral0 que \(M_r'=\frac1n\sum_{i=1}^nX_i^r\). Daí, \[\mathbb{E}M_r'=\mathbb{E}\left[\frac1n\sum_{i=1}^nX_i^r\right]=\frac1n\sum_{i=1}^n\mathbb{E}X_i^r=\mu_r'.\] Por outro lado, \[\mathrm{Var}[M_r']=\mathrm{Var}\left[\frac1n\sum_{i=1}^nX_i^r\right]=\frac{1}{n^2}\sum_{i=1}^n\mathrm{Var}[X_i^r]=\frac1n\left[\mathbb{E}X^{2r}-\left(\mathbb{E}X^r\right)^2\right]. \quad \blacksquare\]

Corolário 2.1 Seja \(X_1,X_2,\ldots,X_n\) uma amostra aleatória de uma população com distribuição com média \(\mu\) e variância \(\sigma^2\). Seja \(\bar{X}_n=\frac1n\sum_{i=1}^nX_i\) a média amostral, então \(\mathbb{E}\bar{X}_n=\mu\) e \(\mathrm{Var}[\bar{X}_n]=\frac{\sigma^2}{n}\).

Definição 2.1 Seja \(X_1,X_2,\ldots,X_n\) uma amostra aleatória de uma população com distribuição que envolve um parâmetro \(\theta\). A variância amostral define-se como: \[S_n^2=\frac{1}{n-1}\sum_{i=1}^n(X_i-\bar{X}_n)^2, \quad n>1.\]

Observação

Em particular, \(M_2=\frac1n\sum_{i=1}^n(X_i-\bar{X}_n)^2\) também é considerada uma medida para quantificar a dispersão. A longo do texto, \(S_n^2\) será considerada a variância amostral porque seu valor esperado é a variância populacional. É importante saber que, quando o tamanho de amostra é suficientemente grande, as propriedades estatísticas de \(M_2\) e \(S^2\) são equivalentes.

Exemplo 2.7 Mostre que \(S_n^{2}=\frac{1}{2n(n-1)} \sum_{i=1}^{n}\sum_{j=1}^{n} (X_{i} - X_{j})^2\), para \(n>1\).

Solução.

Sabemos que, \(S_n^2=\frac{1}{n-1}\sum_{i=1}^n(X_i-\bar{X}_n)^2\), para \(n>1\). Dessa forma, para algum \(X_j\), temos que

\[\begin{align*} (n-1)S_n^2&=\sum_{i=1}^n(X_i-\bar{X}_n)^2=\sum_{i=1}^n(X_i-X_j+X_j-\bar{X}_n)^2\\ &=\sum_{i=1}^n\left[(X_i-X_j)^2+(X_j-\bar{X}_n)^2+2(X_i-X_j)(X_j-\bar{X}_n)\right]. \end{align*}\]

Somando com respeito a \(j\), temos que:

\[\begin{align*} \sum_{j=1}^n(n-1)S_n^2&=n(n-1)S_n^2=\sum_{j=1}^n\sum_{i=1}^n\left[(X_i-X_j)^2+(X_j-\bar{X}_n)^2+2(X_i-X_j)(X_j-\bar{X}_n)\right]\\ &=\sum_{j=1}^n\sum_{i=1}^n(X_i-X_j)^2+\underbrace{\sum_{j=1}^n\sum_{i=1}^n(X_j-\bar{X}_n)^2}_{n(n-1)S_n^2}+2\sum_{j=1}^n\sum_{i=1}^n(X_i-X_j)(X_j-\bar{X}_n). \end{align*}\]

Daí,

\[\begin{align*} \sum_{i=1}^n\sum_{j=1}^n(X_i-X_j)^2&=-2\sum_{i=1}^n\sum_{j=1}^n(X_i-X_j)(X_j-\bar{X}_n) = 2 \sum_{i=1}^{n} \sum_{j=1}^{n} (X_{j} - \bar{X}_n + \bar{X}_n - X_{i})(X_{j} - \bar{X}_n) \\ &= 2 \sum_{i=1}^{n} \sum_{j=1}^{n} (X_{j} - \bar{X}_n)^{2} + 2 \sum_{i=1}^{n} (\bar{X}_n - X_{i}) \underbrace{\sum_{j=1}^{n} (X_{j} - \bar{X}_n)}_{0} = 2n(n - 1)S_n^{2}. \quad \checkmark \end{align*}\]

Teorema 2.2 Seja \(X_1,X_2,\ldots,X_n\) uma amostra aleatória de uma população com distribuição com média \(\mu\) e variância \(\sigma^2\). Considere \(S_n^2\), como na Definição 2.1. Então, \(\mathbb{E}S_n^2=\sigma^2\) e \(\mathrm{Var}[S_n^2]=\frac1n\left(\mu_4-\frac{n-3}{n-1}\sigma^4\right)\), para \(n>1\), onde \(\mu_4=\mathbb{E}(X_i-\mu)^4\) e \(\mathbb{E}X_i=\mu\), para \(i=1,2,3,\ldots,n\).

Prova.

Primeiramente, note que

\[\begin{align*} \sum_{i=1}^n(X_i-\mu)^2&=\sum_{i=1}^n(X_i-\bar{X}_n+\bar{X}_n-\mu)^2= \sum_{i=1}^n(X_i-\bar{X}_n)^2+\sum_{i=1}^n(\bar{X}_n-\mu)^2+2\sum_{i=1}^n(X_i-\bar{X}_n)(\bar{X}_n-\mu)\\ &=\sum_{i=1}^n(X_i-\bar{X}_n)^2+n(\bar{X}_n-\mu)^2+2(\bar{X}_n-\mu)\underbrace{\sum_{i=1}^n(X_i-\bar{X}_n)}_{0}\\ &=\sum_{i=1}^n(X_i-\bar{X}_n)^2+n(\bar{X}_n-\mu)^2. \end{align*}\]

Dessa forma, temos que

\[\begin{align*} \mathbb{E}S_n^2&=\mathbb{E}\left[\frac{1}{n-1}\sum_{i=1}^n(X_i-\bar{X}_n)^2\right] =\frac{1}{n-1}\mathbb{E}\left[\sum_{i=1}^n(X_i-\mu)^2-n(\bar{X}_n-\mu)^2\right]\\ &=\frac{1}{n-1}\left\{\sum_{i=1}^n\mathbb{E}(X_i-\mu)^2-n\mathbb{E}(\bar{X}_n-\mu)^2\right\}= \frac{1}{n-1}\left\{\sum_{i=1}^n\sigma^2-n\frac1n\sigma^2\right\}\\ &=\frac{1}{n-1}\left\{n\sigma^2-\sigma^2\right\}=\sigma^2, \quad n>1. \quad \checkmark \end{align*}\]

Por outra parte, para mostrar \(\mathrm{Var}[S_n^2]=\frac1n\left(\mu_4-\frac{n-3}{n-1}\sigma^4\right)\) considere \(Z_i=X_i-\mu\), para \(i=1,2,\ldots,n\). A sequência \(Z_1,Z_2,\ldots,Z_n\) representa uma amostra aleatória, tal que, para \(i\ne j\ne k\ne l\), temos:

\[\begin{align*} \mathbb{E}[Z_i]&=0; & \mathbb{E}[Z_iZ_j]&=0; & \mathbb{E}[Z_i^3Z_j]&=0; & \mathbb{E}[Z_i^2Z_jZ_k]&=0;\\ \mathbb{E}[Z_i^2]&=\sigma^2; &\mathbb{E}[Z_i^4]&=\mu_4; & \mathbb{E}[Z_i^2Z_j^2]&=\mu_2^2; & \mathbb{E}[Z_iZ_jZ_k,Z_l]&=0. \end{align*}\]

Por outro lado, de acordo com o resultado da Seção A.1 do Apêndice A, temos que:

  • O segundo momento de \(\sum_{i=1}^nZ_i^2\) é dado por

    \[\begin{align*} \mathbb{E}\left[\sum_{i=1}^nZ_i^2\right]^2&=\mathrm{Var}\left[\sum_{i=1}^nZ_i^2\right]+\left(\mathbb{E}\left[\sum_{i=1}^nZ_i^2\right]\right)^2=\sum_{i=1}^n\mathrm{Var}\left[Z_i^2\right]+\left(\sum_{i=1}^n\mathbb{E}\left[Z_i^2\right]\right)^2\\ &=\sum_{i=1}^n\mathbb{E}\left[Z_i^4\right]-\sum_{i=1}^n\left(\mathbb{E}\left[Z_i^2\right]\right)^2+\left(\sum_{i=1}^n\mathrm{Var}\left[Z_i\right]+\sum_{i=1}^n\left(\mathbb{E}[Z_i]\right)^2\right)^2\\ &=n\mu_4-\sum_{i=1}^n\left(\mathrm{Var}[Z_i]\right)^2+\left(\sum_{i=1}^n\mu_2\right)^2=n\mu_4-n\mu_2^2+n^2\mu_2^2=n\mu_4+n(n-1)\mu_2^2. \quad \checkmark \end{align*}\]

  • Também temos que,

    \[\begin{align*} \mathbb{E}\left[\sum_{i=1}^nZ_i\right]^4&=\mathbb{E}\left[\sum_{i=1}^n\sum_{j=1}^nZ_iZ_j\right]^2=\mathbb{E}\left[\sum_{i=1}^n\sum_{j=1}^n\sum_{k=1}^n\sum_{l=1}^nZ_iZ_jZ_kZ_l\right]. \end{align*}\]

    Para calcular a soma, devemos avaliar todos os casos possíveis onde os índices \((i,j,k,l)\) podem tomar diferentes valores:

    • Se houver algum índice diferente de todos os outros índices, e.g, \(i\ne j=k=l\), então o valor esperado é zero;

    • Se todos os índices forem iguais, i.e. \(i=j=k=l\), então \(\mathbb{E}Z_i^4=\mu_4\). Ressaltando que existem \(n\) maneiras de isso acontecer;

    • Se tiver dois pares distintos de índices iguais, e.g, \(i=j\ne k=l\), nesse caso, suponha que \(i=j\) e \(k=l\), com \(i<k\), então \[\mathbb{E}[Z_i^2Z_k^2]=\mathbb{E}[Z_i^2]\mathbb{E}[Z_k^2]=\mu_2^2.\] Ressaltando que existem \(\binom{n}{2}\binom{4}{2}=\frac{n!}{(n-2)!2!}\frac{4!}{2!2!}=3n(n-1)\) maneiras de acontecer dois pares de índices iguais.

    Dessa forma, temos que

    \[\begin{align*} \mathbb{E}\left[\sum_{i=1}^nZ_i\right]^4&=\mathbb{E}\left[\sum_{i=1}^n\sum_{j=1}^nZ_iZ_j\right]^2=n\mu_4+3n(n-1)\mu_2^2. \quad \checkmark \end{align*}\]

  • Ainda, temos que

    \[\begin{align*} \mathbb{E}\left[\left(\sum_{i=1}^nZ_i^2\right)\left(\sum_{i=1}^nZ_i\right)^2\right]&=\mathbb{E}\left[\left(\sum_{i=1}^nZ_i^2\right)\left(\sum_{i=1}^n\sum_{j=1}^nZ_iZ_j\right)\right]=\mathbb{E}\left[\sum_{i=1}^n\sum_{j=1}^n\sum_{k=1}^nZ_i^2Z_jZ_k\right]. \end{align*}\]

    De igual forma que no caso anterior, devem ser avaliados os casos onde os índices \((i,j,k)\) podem tomar valores diferentes:

    • Para \(j\ne k\), o valor esperado é \(0\);
    • Quando \(i=j=k\), então \(\mathbb{E}Z_i^4=\mu_4\) e isso acontece de \(n\) maneiras;
    • Quando \(i\ne j=k\), o valor esperado é \(\mu_2^2\), mas note que agora a ordem \(i,j\) importa, então, temos \(n(n-1)\) formas de obter esse valor esperado.

    Por tanto,

    \[\begin{align*} \mathbb{E}\left[\left(\sum_{i=1}^nZ_i^2\right)\left(\sum_{i=1}^nZ_i\right)^2\right]&=\mathbb{E}\left[\sum_{i=1}^n\sum_{j=1}^n\sum_{k=1}^nZ_i^2Z_jZ_k\right]=n\mu_4+n(n-1)\mu_2^2. \quad \checkmark \end{align*}\]

Finalmente, para calcular a variância de \[S_n^2=\frac{1}{n-1}\sum_{i=1}^n(X_i-\bar{X}_n)^2=\frac{n\sum_{i=1}^nZ_i^2-\left(\sum_{i=1}^nZ_i\right)^2}{n(n-1)},\] sabemos que \(\mathrm{Var}[S_n^2]=\mathbb{E}S_n^4-\left(\mathbb{E}S_n^2\right)^2=\mathbb{E}S_n^4-\mu_2^2\), onde

\[\begin{align*} \mathbb{E}\left[S_n^4\right]&=\frac{n^2\mathbb{E}\left(\sum_{i=1}^nZ_i^2\right)^2-2n\mathbb{E}\left[\left(\sum_{i=1}^nZ_i^2\right)\left(\sum_{i=1}^nZ_i\right)^2\right]+\mathbb{E}\left[\sum_{i=1}^nZ_i\right]^4}{n^2(n-1)^2}\\ &=\frac{n^2(n\mu_4+n(n-1)\mu_2^2)-2n(n\mu_4+n(n-1)\mu_2^2)+n\mu_4+3n(n-1)\mu_2^2}{n^2(n-1)^2}\\ &=\frac{(n^3-2n^2+n)\mu_4+(n^3(n-1)-2n^2(n-1)+3n(n-1))\mu_2^2}{n^2(n-1)^2}\\ &=\frac{n(n-1)^2\mu_4+n(n-1)(n^2-2n+3)\mu_2^2}{n^2(n-1)^2}=\frac{(n-1)\mu_4+(n^2-2n+3)\mu_2^2}{n(n-1)}. \end{align*}\]

Daí,

\[\begin{align*} \mathrm{Var}\left[S_n^2\right]&=\frac{(n-1)\mu_4+(n^2-2n+3)\mu_2^2}{n(n-1)}-\mu_2^2=\frac{(n-1)\mu_4+(n^2-2n+3)\mu_2^2-n(n-1)\mu_2^2}{n(n-1)}\\ &=\frac{(n-1)\mu_4-(n-3)\mu_2^2}{n(n-1)}=\frac1n\left\{\mu_4-\frac{n-3}{n-1}\mu_2^2\right\}. \quad\blacksquare \end{align*}\]

2.2 Funções geradoras de momentos

Definição 2.2 (Função geradora de momentos) Seja \(X\) uma variável aleatória de uma população com função de distribuição \(F_X\). A função geradora de momentos (FGM) de \(F_X\), define-se como \[M_X(t)=\mathbb{E}\left[e^{tX}\right],\] se existe para todo \(t\), tal que \(-h<t<h\), para algum \(h>0\).

Teorema 2.3 Seja \(M_X(t)\) a FGM de uma distribuição \(F_X\). Se existe \(M_X(t)\) para todo \(|t|<h\), para algum \(h>0\), então \(\mathbb{E}X^r\), \(r=1,2,\ldots\), existe e

\[\mathbb{E}X^r=M_X^{(r)}(0)=\frac{\partial^r}{\partial t^r}M_X(t)\Biggr|_{t=0}.\]

Prova.

Pela expansão em série de Taylor da função exponencial (vide Apêndice A), temos que \[e^x=1+x+\frac{1}{2!}x^2+\frac{1}{3!}x^3+\frac{1}{4!}x^4+\cdots.\]

Por outra parte, pela Definição 2.2, temos que \(M_X(t)=\mathbb{E}\left[e^{tX}\right]\), então

\[\begin{align*} M_X(t)&=\mathbb{E}\left[e^{tX}\right]=\mathbb{E}\left[\sum_{j=0}^\infty\frac{1}{j!}(tX)^j\right] =\sum_{j=0}^\infty\frac{t^j}{j!}\mathbb{E}\left[X^j\right]. \end{align*}\]

Por hipóteses, \(\mathbb{E}X^r\), \(r=1,2,\ldots\), existem para todo \(|t|<h\), para algum \(h>0\).

Diferenciando em ambos os lados da igualdade e avaliando em \(t=0\), temos que

\[\begin{align*} M'_X(0)&=\left(0+\mathbb{E}X+\mathbb{E}X^2t+\cdots+\mathbb{E}\left[X^r\right]\frac{t^{r-1}}{(r-1)!}\right)\Biggr|_{t=0}=\mathbb{E}X. \quad\checkmark \end{align*}\]

Em geral,

\[\begin{align*} M_X^{(r)}(0)&=\frac{\partial^r}{\partial t}M_x(t)\Biggr|_{t=0}=\mathbb{E}\left[\frac{\partial}{\partial t}\frac{\partial^{r-1}}{\partial t^{r-1}}e^{tX}\right]\Biggr|_{t=0}=\mathbb{E}\left[X^re^{tX}\right]\Biggr|_{t=0}=\mathbb{E}X^r. \quad\blacksquare \end{align*}\]

Exemplo 2.8 Seja \(X\) uma variável aleatória de uma população com distribuição gama com parâmetros \(\alpha\) e \(\beta\), i.e. \[f(x; \alpha, \beta)=\frac{1}{\Gamma(\alpha)}\beta^\alpha x^{\alpha-1}e^{-\beta x}\mathbb{I}_{(0,\infty)}(x).\] Utilize a FGM para calcular a variância de \(X\).

Solução.

Pela definição Definição 2.2, sabemos que \(M_X(t)=\mathbb{E}\left[e^{tX}\right]\), para todo \(t\), tal que \(|t|<h\), para algum \(h>0\). Dessa forma,

\[\begin{align*} M_X(t)&=\int_0^\infty e^{tx}\frac{1}{\Gamma(\alpha)}\beta^\alpha x^{\alpha-1}e^{-\beta x}\,dx =\frac{1}{\Gamma(\alpha)}\beta^\alpha\int_0^\infty x^{\alpha-1}e^{-(\beta-t)x}\,dx. \end{align*}\]

Para avaliar a existência da integral, consideremos os seguintes casos:

  • Se \(t=\beta\),
\[\begin{align*} M_X(t)&=\frac{1}{\Gamma(\alpha)}\beta^\alpha\int_0^\infty x^{\alpha-1}e^{-(\beta-t)x}\,dx=\frac{\beta^\alpha}{\Gamma(\alpha)}\lim_{a\to\infty}\int_0^a x^{\alpha-1}\,dx=\frac{\beta^\alpha}{\alpha\Gamma(\alpha)}\lim_{a\to\infty} a^{\alpha}. \end{align*}\]

Para \(\alpha>0\), \(\lim_{a\to\infty} a^{\alpha}\) é infinito. Dessa forma, \(M_X(t)\) não está definida.

  • Se \(t>\beta\), a quantidade \(-(\beta-t)>0\), então \(e^{-(\beta-t)x}>x^{\alpha-1}\). Assim, a integral é divergente e, por conseguinte, a FGM não existe.

  • Se \(t<\beta\),

\[\begin{align*} M_X(t)&=\frac{1}{\Gamma(\alpha)}\beta^\alpha\int_0^\infty x^{\alpha-1}e^{-(\beta-t)x}\,dx =\frac{\beta^\alpha}{\Gamma(\alpha)}\frac{1}{(\beta-t)^\alpha}\int_0^\infty u^{\alpha-1}e^{-u}\,du, \quad u=(\beta-t)x\\ &=\frac{\beta^\alpha}{\Gamma(\alpha)}\frac{1}{(\beta-t)^\alpha}\Gamma(\alpha) =\frac{\beta^\alpha}{(\beta-t)^\alpha}=\left(1-\frac{t}{\beta}\right)^{-\alpha}. \quad\checkmark \end{align*}\]

Para calcular a variância, utilizamos o resultado do Teorema 2.3. Daí

\[\begin{align*} M'_X(0)&=\frac{\alpha}{\beta}\left(1-\frac{t}{\beta}\right)^{-\alpha-1}\Biggr|_{t=0}=\frac{\alpha}{\beta}\\ M''_X(0)&=\frac{\alpha(\alpha+1)}{(\beta-x)^2}\left(1-\frac{t}{\beta}\right)^{-\alpha}\Biggr|_{t=0} =\frac{\alpha(\alpha+1)}{\beta^2}. \end{align*}\]

Dessa forma,

\[\begin{align*} \mathrm{Var}[X]&=\mathbb{E}X^2-\left(\mathbb{E}X\right)^2=\frac{\alpha(\alpha+1)}{\beta^2}-\left(\frac{\alpha}{\beta}\right)^2=\frac{\alpha}{\beta^2}.\quad\checkmark \end{align*}\]

Exemplo 2.9 Seja \(X_1,X_2\) uma amostra aleatória de uma população com densidade conjunta dada por \[f(x; a)=a^2e^{-ax_2}, \quad 0<x_1<x_2<\infty, \quad a>0.\] Encontre \(\mathbb{E}X^r_1\), \(\mathbb{E}X^r_2\), para \(r=1,2,3,\ldots\), e \(\mathrm{Corr}[X_1,X_2]\).

Solução.

Temos que,

\[\begin{align*} \mathbb{E}X^r_1&=\int_0^\infty\int_0^{x_2}x_1^ra^2e^{-ax_2}\,dx_1dx_2 =\int_0^\infty a^2e^{-ax_2}\int_0^{x_2}x_1^r\,dx_1dx_2\\ &=\int_0^\infty a^2e^{-ax_2}\frac{x_1^{r+1}}{r+1}\Biggr|_{0}^{x_2}\,dx_2 =\int_0^\infty a^2e^{-ax_2}\frac{x_2^{r+1}}{r+1}\,dx_2\\ &=\frac{\Gamma(r+1)}{a^r}=\frac{r!}{a^r}, \quad r=1,2,\ldots. \quad\checkmark \end{align*}\]

Por outro lado, temos que

\[\begin{align*} \mathbb{E}X^r_2&=\int_0^\infty\int_0^{x_2}x_2^ra^2e^{-ax_2}\,dx_1dx_2 =\int_0^\infty a^2x_2^re^{-ax_2}\int_0^{x_2}\,dx_1dx_2\\ &=\int_0^\infty a^2x_2^{r+1}e^{-ax_2}\,dx_2=\frac{\Gamma(r+2)}{a^r}=\frac{(r+1)!}{a^r}, \quad r=1,2,\ldots. \quad\checkmark \end{align*}\]

O momento cruzado é calculado como

\[\begin{align*} \mathbb{E}X_1X_2&=\int_0^\infty\int_0^{x_2}x_1x_2a^2e^{-ax_2}\,dx_1dx_2 =\int_0^\infty a^2x_2e^{-ax_2}\int_0^{x_2}x_1\,dx_1dx_2\\ &=\frac{a^2}{2}\int_0^\infty x_2^3e^{-ax_2}dx_2 =\frac{\Gamma(4)}{2a^2}\int_0^\infty \frac{a^4}{\Gamma(4)}x_2^3e^{-ax_2}dx_2=\frac{3}{a^2}. \quad\checkmark \end{align*}\]

Dessa forma, a correlação é dada por

\[\begin{align*} \mathrm{Corr}[X_1,X_2]&=\frac{\mathbb{E}X_1X_2-\mathbb{E}X_1\mathbb{E}X_2}{\sqrt{\mathrm{Var}[X_1]\mathrm{Var}[X_2]}} =\frac{\frac{3}{a^2}-\frac1a\frac2a}{\sqrt{\frac{1}{a^2}\frac{2}{a^2}}}=\frac{1}{\sqrt{2}}. \quad\checkmark \end{align*}\]

Exemplo 2.10 Seja \(X\) uma variável aleatória de uma população com distribuição geométrica com parâmetro \(p\), i.e.. \(\mathbb{P}[X=x]=p(1-p)^x\mathbb{I}_{\{0,1,2,\ldots\}}(x)\). Apresente uma forma analítica para calcula o \(r\)-ésimo momento da distribuição de \(X\).

Solução.

A FGM é dada por

\[\begin{align*} M_X(t)&=\mathbb{E}\left[e^{tX}\right]=\sum_{x=0}^\infty e^{tx}p(1-p)^x=p\sum_{x=0}^\infty \left(e^{t}(1-p)\right)^x=\frac{p}{1-(1-p)e^t}. \end{align*}\]

A última soma pode ser tratada como uma série geométrica. Nesse caso, a soma converge quando \((1-p)e^t<1\) ou \(t<-\log(1-p)\).

Daí,

\[\begin{align*} \mathbb{E}\left[X\right]&=\frac{p(1-p)e^t}{(1-(1-p)e^t)^2}\Biggr|_{t=0}=\frac{1-p}{p};\\ \\ \mathbb{E}\left[X^2\right]&=\frac{p(1-p)e^t(1+(1-p)e^t)}{(1-(1-p)e^t)^3}\Biggr|_{t=0}=\frac{(1-p)(2-p)}{p^2}. \end{align*}\]

A variância é dada por \(\mathrm{Var}[X]=\mathbb{E}X^2-(\mathbb{E}X)^2=\dfrac{(1-p)(2-p)}{p^2}-\dfrac{(1-p)^2}{p^2}=\dfrac{1-p}{p^2}\).

O cálculo das derivadas de ordem superior tornam trabalhoso o uso dessa estratégia. Uma forma analítica para o \(r\)-ésimo momento pode ser obtida como

\[\begin{align*} \mathbb{E}\left[X^r\right]&=\sum_{x=0}^\infty x^r(1-q)q^x=\sum_{x=0}^\infty x^{r-1}(1-q)qxq^{x-1}=(1-q)q\sum_{x=0}^\infty x^{r-1}xq^{x-1}\\ &=(1-q)q\sum_{x=0}^\infty x^{r-1}\frac{\partial}{\partial q}q^{x}=(1-q)q\frac{\partial}{\partial q}\left[\sum_{x=0}^\infty x^{r-1}q^{x}\right]=(1-q)q\frac{\partial}{\partial q}\left[\frac{1}{1-q}\sum_{x=0}^\infty x^{r-1}(1-q)q^{x}\right]\\ &=(1-q)q\frac{\partial}{\partial q}\left[\frac{1}{1-q}\mathbb{E}X^{r-1}\right], \end{align*}\]

onde \(q=1-p\) e \(r=1, 2, \ldots\). \(\quad\checkmark\)

Exemplo 2.11 Seja \(X\) uma variável aleatória de uma população com distribuição Pareto com parâmetros \(\alpha>0\) e \(\beta>0\), i.e. \[f(x;\alpha,\beta)=\beta\frac{\alpha^\beta}{x^{\beta+1}}\mathbb{I}_{[\alpha,\infty)}(x).\] Utilize a FGM da distribuição de \(X\) para calcular a sua variância.

Solução.

Pela Definição 2.2, temos que

\[\begin{align*} M_X(t)&=\beta\alpha^\beta\int_\alpha^\infty\frac{e^{tx}}{x^{\beta+1}}\,dx. \end{align*}\]

Note que, quando \(x\to\infty\), \(\frac{e^{tx}}{x^{\beta+1}}\to\infty\) fazendo com que a FGM não exista.

No entanto, pode-se observar que para alguns valores específicos de \(\beta\), alguns momentos da distribuição de \(X\) podem existir. Por exemplo, considere-se o caso \(\beta>r\), daí

\[\begin{align*} \mathbb{E}X^r&=\beta\alpha^\beta\int_\alpha^\infty \frac{x^r}{x^{\beta+1}}\,dx =\beta\alpha^\beta\int_\alpha^\infty x^{r-\beta-1}\,dx\\ &=-\beta\alpha^\beta\frac{x^{-\beta+r}}{\beta-r}\Biggr|_\alpha^\infty=\frac{\beta\alpha^r}{\beta-r}. \quad\checkmark \end{align*}\]

Em particular, se \(\beta>2\), temos que

\[\begin{align*} \mathbb{E}X&=\frac{\beta\alpha}{\beta-1} & &\mathrm{ e } & \mathbb{E}X^2&=\frac{\beta\alpha^2}{\beta-2}. \end{align*}\]

A variância de \(X\) é dada por

\[\mathrm{Var}[X]=\frac{\beta\alpha^2}{\beta-2}-\left(\frac{\beta\alpha}{\beta-1}\right)^2=\frac{\alpha\beta^2}{(\beta-1)^2(\beta-2)}. \quad\checkmark\]

Dica

Seja \(Y=a+bX\), onde \(X\) é uma variável aleatória, \(a\) e \(b\) constantes, então \(M_Y(t)=\mathbb{E}\left[e^{at}e^{btX}\right]=e^{at}\mathbb{E}\left[e^{btX}\right]=e^{at}M_X(bt)\), com \(|t|<\frac{h}{|b|}\), \(h>0\).

Exemplo 2.12 Seja \(X\) uma variável aleatória de uma população com distribuição uniforme no intervalo \((0,1)\), i.e. \(f(x)=\mathbb{I}_{(0,1)}(x)\). Seja \(y=(b-a)X+a\), onde \(a,b\) são constantes, tal que \(b>a\). Encontre a FGM da distribuição de \(Y\).

Solução.

A FGM da distribuição de \(X\) é dada por

\[M_X(t)=\mathbb{E}[e^{tX}]=\int_0^1e^{tx}\,dx=\frac{e^t-1}{t}, \quad t\ne0.\] Então, a FGM da distribuição de \(Y\) é dada por

\[M_Y(t)=e^{at}\frac{e^{(b-a)t}-1}{(b-a)t}=\frac{e^{bt}-e^{at}}{(b-a)t}, \quad t\ne0. \quad\checkmark\]

\(Y\) é uma variável aleatória de uma população com distribuição uniforme no intervalo \((a,b)\). O \(r\)-ésimo momento da distribuição de \(Y\) é dado por \[\mathbb{E}Y^r=\int_a^b\frac{y^r}{b-a}\,dy=\frac{b^{r+1}-a^{r+1}}{(n+1)(b-a)}.\]

Teorema 2.4 Sejam \(X\) e \(Y\) duas variáveis aleatórias com FGM \(M_{X}(t)\) e \(M_{Y}(t)\), respectivamente. Se existe \(h>0\), tal que \(M_{X}(t)=M_{Y}(t)\), para todo \(|t|<h\), então \(F_X(u)=F_Y(u)\) para todo \(u\in\mathbb{R}\).

A prova do Teorema 2.4 requer alguns conceitos que extrapolam o nivel do conteúdo. Para os detalhes sugere-se, e.g., Billingsley (1986) e Curtiss (1942).

Problema dos momentos

Seja \(X\) uma variável aleatória de uma população com distribuição \(F(x)\). Suponha que todos os momentos de \(F(x)\) existem, i.e. \(\mathbb{E}X\), \(\mathbb{E}X^2\), \(\mathbb{E}X^3\ldots\) existem. A distribuição de \(X\) pode ser determinada de forma única pelos seus momentos?

Muitas vezes refere-se a esse questionamento como Problema dos momentos de Hausdorff, em honor ao matemático alemão Felix Hausdorff (1868-1942). Em \(1921\), Hausdorff mostrou as condições necessárias e suficientes para determinar de forma única a distribuição de uma variável aleatória a partir dos seus momentos populacionais em torno de zero. Na literatura, muitos autores discutem amplamente os detalhes desse problema e as propostas para sua solução. Sugere-se a leitura, e.g., Dudewicz e Mishra (1988) [pp. 261], Shohat e Tamarkin (1970) e Feller (1971, pp. 224).

Teorema 2.5 Seja \(X_1,X_2,\ldots,X_n\) uma amostra aleatória de uma população com função de distribuição \(F\). Seja \(Y=\sum_{i=1}^na_iX_i\), com \(a_1,a_2,\ldots,a_n\) constantes. A FGM da distribuição de \(Y\) é dada por \[M_{Y}(t)=\prod_{i=1}^nM_{X_1}(a_it),\] para todo \(|t|<h\), , para algum \(h>0\).

Prova

O resultado é consequência da Definição 2.2 e do fato que as variáveis aleatórias \(X_1,X_2,\ldots,X_n\) são independentes e identicamente distribuídas. Assim,

\[\begin{align*} M_Y(t)&=\mathbb{E}\left[e^{tY}\right]=\mathbb{E}\left[e^{t\sum_{i=1}^na_iX_i}\right] =\mathbb{E}\left[\prod_{i=1}^ne^{a_itX_i}\right] =\prod_{i=1}^n\mathbb{E}\left[e^{a_itX_i}\right]=\prod_{i=1}^nM_{X_1}(a_it).\quad\blacksquare \end{align*}\]

Se \(a_1=a_2=\cdots=a_n\), então \(M_Y(t)=\left[M_{X_1}(a_it)\right]^n\).

Exemplo 2.13 Seja \(X_1,X_2,\ldots,X_n\) uma amostra aleatória de uma população com distribuição normal com média \(\mu\) e variância \(\sigma^2\). Mostre que \(\bar{X}_n\) tem distribuição normal com média \(\mu\) e variância \(\frac{\sigma^2}{n}\).

Solução.

Aplicando o resultado do Teorema 2.5, temos que

\[\begin{align*} M_{\bar{X}_n}(t)&=\mathbb{E}\left[e^{t\frac1n\sum_{i=1}^nX_i}\right] =\left[M_{X_1}\left(\frac{t}{n}\right)\right]^n =\left[\exp\left\{\frac{\mu}{n}t+\frac12\left(\frac{\sigma t}{n}\right)^2\right\}\right]^n =\exp\left\{\mu t+\frac{1}{2n}\left(\sigma t\right)^2\right\}, t\in\mathbb{R}.\quad\checkmark \end{align*}\]

Exemplo 2.14 Seja \(X_1,X_2,\ldots,X_n\) uma amostra aleatória de uma população com distribuição Poisson com parâmetro \(\lambda\), i.e. \(\mathbb{P}[X=x]=\frac{e^{-\lambda}\lambda^x}{x!}\mathbb{I}_{\{0,1,2,\ldots\}}(x)\). Calcule a FGM da distribuição de \(Y=\sum_{i=1}^nX_i\).

Solução.

A FGM é dada por

\[\begin{align*} M_X(t)&=\mathbb{E}\left[e^{tX}\right]=\sum_{x=0}^\infty e^{tx} \frac{e^{-\lambda}\lambda^x}{\lambda!}=e^{-\lambda}\sum_{x=0}^\infty \frac{(\lambda e^t)^x}{\lambda!}=e^{-\lambda}e^{\lambda e^{t}}=e^{\lambda(e^t-1)}, \quad t\in\mathbb{R}. \end{align*}\]

Daí, pelo Teorema 2.5, temos que

\[\begin{align*} M_Y(t)&=\left(e^{\lambda(e^t-1)}\right)^n, \quad t\in\mathbb{R}. \end{align*}\]

Dessa forma, \(Y\) é uma variável aleatória de uma população com distribuição Poisson com parâmetro \(n\lambda\).

Teorema 2.6 Seja \(X_1,X_2,\ldots,X_k\) uma sequência de variáveis aleatórias independentes com distribuição normal com média \(\mu_i\) e variância \(\sigma^2_i\), para \(i=1,2,\ldots, k\). Então \[U=\sum_{i=1}^k\left(\frac{X_i-\mu_i}{\sigma_i}\right)^2,\] segue uma distribuição \(\chi^2_{(k)}\).

Prova.

Seja \(Z_i=\frac{X_i-\mu_i}{\sigma_i}\). Dessa forma, a sequência \(Z_1,Z_2,\ldots,Z_n\) configura uma amostra aleatória de uma população com distribuição normal padrão. Daí,

\[\begin{align*} \mathbb{E}\left[e^{tZ^2_1}\right]&=\int_{-\infty}^\infty e^{tz^2}\frac{1}{\sqrt{2\pi}}e^{-\frac12z^2}\,dz = \frac{1}{\sqrt{2\pi}}\int_{-\infty}^\infty e^{-\frac12(1-2t)z^2}\,dz\\ &= \frac{1}{\sqrt{2\pi}}\frac{\sqrt{1-2t}}{\sqrt{1-2t}}\int_{-\infty}^\infty e^{-\frac12(1-2t)z^2}\,dz = \frac{1}{\sqrt{1-2t}}, \quad t<\frac12. \end{align*}\]

Pelo Teorema 2.5, a FGM da distribuição de \(U\) é dada por

\[\begin{align*} M_U(t)&=\left(\frac{1}{\sqrt{1-2t}}\right)^k, \quad t<\frac12. \quad\checkmark \end{align*}\]

Seguindo o resultado do Exemplo 2.8, se \(\alpha=\frac{k}{2}\) e \(\beta=\frac12\), então \[M_X(t)=\left(1-2t\right)^{-\frac{k}{2}}, \quad t<\frac12.\] Sendo a FGM de uma variável aleatória com distribuição \(\chi^2_{(k)}\). Dessa forma, pelo Teorema 2.4, a distribuição da variável aleatória \(U\) é \(\chi^2_{(k)}\).

Corolário 2.2 Seja \(X_1,X_2,\ldots,X_k\) uma amostra aleatória de uma população com distribuição normal com média \(\mu\) e variância \(\sigma^2\). Então, \[U=\sum_{i=1}^k\left(\frac{X_i-\mu}{\sigma}\right)^2\] é uma variável aleatória de uma população com distribição \(\chi^2_{(k)}\).

Exemplo 2.15 Seja \(X_1,X_2,\ldots,X_n\) uma amostra aleatória de uma população com distribuição normal com média \(\mu\) e variância \(\sigma^2\). Sejam \(\bar{X}_n=\frac1n\sum_{i=1}^nX_i\) e \(S^2_n=\frac{1}{n-1}\sum_{i=1}^n(X_i-\bar{X}_n)^2\). Mostre que \(\frac{(n-1)S^2_n}{\sigma^2}\) segue uma distribuição \(\chi^2_{(n-1)}\).

Solução.

Sabemos que \(S^2_n=\frac{1}{n-1}\sum_{i=1}^n(X_i-\bar{X}_n)^2\), daí

\[\begin{align*} (n-1)S^2_n&=\sum_{i=1}^n(X_i-\bar{X}_n)^2=\sum_{i=1}^{n-1}(X_i-\bar{X}_n)^2+(X_n-\bar{X}_n)^2\\ &=\sum_{i=1}^{n-1}(X_i-\bar{X}_{n-1}+\bar{X}_{n-1}-\bar{X}_n)^2+(X_n-\bar{X}_n)^2\\ &=\sum_{i=1}^{n-1}(X_i-\bar{X}_{n-1})^2+\sum_{i=1}^{n-1}(\bar{X}_{n-1}-\bar{X}_n)^2+2(\bar{X}_{n-1}-\bar{X}_n)\underbrace{\sum_{i=1}^{n-1}(X_i-\bar{X}_{n-1})}_{0}+(X_n-\bar{X}_n)^2\\ &=\sum_{i=1}^{n-1}(X_i-\bar{X}_{n-1})^2+(n-1)(\bar{X}_{n-1}-\bar{X}_n)^2+(X_n-\bar{X}_n)^2\\ &=(n-2)S^2_{n-1}+(n-1)\left(\frac{n}{n(n-1)}\sum_{i=1}^{n-1}X_i-\frac{n-1}{n(n-1)}\sum_{i=1}^{n}X_i\right)^2+(X_n-\bar{X}_n)^2\\ &=(n-2)S^2_{n-1}+\frac{1}{n^2(n-1)}\left(nX_n-\sum_{i=1}^{n}X_i\right)^2+(X_n-\bar{X}_n)^2\\ &=(n-2)S^2_{n-1}+\frac{1}{n-1}\left(X_n-\bar{X}_n\right)^2+(X_n-\bar{X}_n)^2 =(n-2)S^2_{n-1}+\frac{n}{n-1}\left(X_n-\bar{X}_n\right)^2\\ &=(n-2)S^2_{n-1}+\frac{n-1}{n}\left(X_n-\bar{X}_{n-1}\right)^2, \quad n>1.\quad\checkmark \end{align*}\]

Dessa forma, aplicando indução matemática, temos que

  • Para \(n=2\), temos que \(S^2_2=\frac{1}{2}\left(X_2-X_1\right)^2\). Sendo \(X_1,X_2,\ldots,X_n\) uma amostra aleatória de uma população com distribuição normal com média \(\mu\) e variância \(\sigma^2\), então \(X_2-X_1\) segue uma distribuição normal com média \(0\) e variância \(2\sigma^2\). Daí, \(\frac{\left(X_2-X_1\right)^2}{2\sigma^2}=\frac{S_2^2}{\sigma^2}\) segue uma distribuição \(\chi^2_{(1)}\);

  • Suponha que, para \(n=k\), \(\frac{(k-1)S^2_k}{\sigma^2}\) segue uma distribuição \(\chi^2_{(k-1)}\);

  • Provemos para \(n=k+1\),

\[\begin{align*} \frac{kS^2_{k+1}}{\sigma^2}&=\frac{(k-1)S^2_{k}}{\sigma^2}+\frac{1}{\sigma^2}\frac{k}{k+1}\left(X_{k+1}-\bar{X}_{k}\right)^2. \end{align*}\]

Observe que, \(X_{k+1}-\bar{X}_{k}\) segue uma distribuição normal com média \(0\) e variância \(\left(\frac{k+1}{k}\right)\sigma^2\). Daí, \(\frac{k}{k+1}\frac{(X_{k+1}-\bar{X}_{k})^2}{\sigma^2}\) segue uma distribuição \(\chi^2_{(1)}\). Pelo Teorema 2.6, \(\frac{kS^2_{k+1}}{\sigma^2}\) segue uma distribuição \(\chi^2_{(k)}\). \(\quad\checkmark\)

Teorema 2.7 Seja \(X_1,X_2,\ldots,X_n\) uma amostra aleatória de uma população com distribuição normal com média \(\mu\) e variância \(\sigma^2\). Sejam \(\bar{X}_n=\frac1n\sum_{i=1}^nX_i\) e \(S^2_n=\frac{1}{n-1}\sum_{i=1}^n\left(X_i-\bar{X}_n\right)^2\), então

  1. \(\bar{X}_n\) segue uma distribuição normal com média \(\mu\) e variância \(\frac{\sigma^2}{n}\);
  2. \(\frac{(n-1)S^2_n}{\sigma^2}\) segue uma distribuição \(\chi^2_{(n-1)}\);
  3. \(\bar{X}_n\) e \(S^2_n\) são variáveis aleatórias independentes.
Prova.
  1. Vide Teorema 2.1 e Exemplo 2.13;
  2. Vide Exemplo 2.15;

  3. Note que

    \[\begin{align*} S^2_n&=\frac{1}{n-1}\sum_{i=1}^n\left(X_i-\bar{X}_n\right)^2 =\frac{1}{n-1}\left[(X_1-\bar{X}_n)+\sum_{i=2}^n\left(X_i-\bar{X}_n\right)^2\right]\\ &=\frac{1}{n-1}\left\{\left[\sum_{i=1}^n(X_i-\bar{X}_n)-(X_1-\bar{X}_n)\right]^2+\sum_{i=2}^n\left(X_i-\bar{X}_n\right)^2\right\}\\ &=\frac{1}{n-1}\left\{\left[\sum_{i=2}^n(X_i-\bar{X}_n)\right]^2+\sum_{i=2}^n\left(X_i-\bar{X}_n\right)^2\right\} \end{align*}\]

    Daí, \(S^2_n\) pode ser escrito em função de \(X_2-\bar{X}_n, X_3-\bar{X}_n, \ldots, X_n-\bar{X}_n\). Dessa forma, se as variáveis aleatórias \(X_2-\bar{X}_n, X_3-\bar{X}_n,\ldots,X_n-\bar{X}_n\) são independentes de \(\bar{X}_n\), obtem-se o resultado requerido.

    Para isso, calcula-se a densidade conjunta das variáveis aleatórias \(\bar{X}_n\), \(X_2-\bar{X}_n, X_3-\bar{X}_n,\ldots,X_n-\bar{X}_n\) para verificar se a mesma pode ser escrita como produto das densidades de \(\bar{X}_n\) e \(X_2-\bar{X}_n, X_3-\bar{X}_n,\ldots,X_n-\bar{X}_n\). Dessa maneira, para usar o método do jacobiano, considere \(y_1=\bar{x}_n\) e \(y_i=x_i-\bar{x}_n\) para \(i=2,3,\ldots,n\). Então,

    \[\begin{align*} x_1&=y_1-\sum_{i=2}^ny_i; & x_i&=y_i+y_1, \quad i=2,3,4,\ldots. \end{align*}\]
    \[\begin{align*} \frac{\partial x_1}{\partial y_1}&=1; & \frac{\partial x_2}{\partial y_1}&=1; & \frac{\partial x_3}{\partial y_1}&=1; & \cdots & & \frac{\partial x_n}{\partial y_1}&=1;\\ \frac{\partial x_1}{\partial y_2}&=-1; & \frac{\partial x_2}{\partial y_2}&=1; & \frac{\partial x_3}{\partial y_2}&=0; & \cdots & & \frac{\partial x_n}{\partial y_2}&=0;\\ \frac{\partial x_1}{\partial y_3}&=-1; & \frac{\partial x_2}{\partial y_3}&=0; & \frac{\partial x_3}{\partial y_3}&=1; & \cdots & & \frac{\partial x_n}{\partial y_3}&=0;\\ &\,\,\,\vdots &&\,\,\,\vdots &&\,\,\,\vdots & & &&\,\,\,\vdots\\ \frac{\partial x_1}{\partial y_3}&=-1; & \frac{\partial x_2}{\partial y_3}&=0; & \frac{\partial x_3}{\partial y_3}&=0; & \cdots & & \frac{\partial x_n}{\partial y_3}&=1. \end{align*}\]

    Assim sendo,

    \[\begin{align*} |J|&=\begin{vmatrix} 1 & 1 & 1 & \cdots & 1\\ -1 & 1 & 0 & \cdots & 0\\ -1 & 0 & 1 & \cdots & 0\\ \vdots & & & \ddots & \vdots \\ -1 & 0 & 0 & \cdots & 1\\ \end{vmatrix}=n. \end{align*}\]

    Por otro lado, sabemos que a função de densidade conjunta de \(X_1,X_2,\ldots,X_n\) é dada por

    \[f(x_1,x_2,\ldots,x_n)=(2\pi)^{-n/2}e^{-\frac12\sum_{i=1}^nx_i}, \quad x_i\in\mathbb{R}.\] Portanto,

    \[\begin{align*} f(y_1,y_2,\ldots,y_n)&=\frac{n}{(2\pi)^{-n/2}}e^{-\frac12(y_1-\sum_{i=2}^ny_i)^2}e^{-\frac12\sum_{i=2}^n(y_i+y_1)^2}\\ &=\frac{1}{(2\pi n^{-1})^{\frac12}}e^{-\frac12ny_1^2}\frac{1}{(2\pi )^{\frac{n-1}{2}}n^{-1/2}}e^{\left[-\frac12\sum_{i=2}^ny_i^2+(\sum_{i=2}^ny_i)^2\right]}. \end{align*}\]
    Então, \(Y_1\) é independente de \(Y_2,Y_3,\ldots,Y_n. \quad \blacksquare\)

2.2.1 Momentos fatoriais

Definição 2.3 (Função geradora de momentos fatoriais) Seja \(X\) uma variável aleatória de uma população com função de distribuição \(F_X\). A função geradora de momentos fatoriais (FGMF) de \(F_X\), denotada por \(\mathcal{M}_X(t)\), se existe, define-se como

\[\mathcal{M}_X(t)=M_X(\log t)=\mathbb{E}\left[t^{X}\right],\]

para todo \(t\in\mathbb{R}\).

Se \(\mathcal{M}_X(t)\) existe em uma vizinhança de \(t = 1\), o \(r\)-ésimo momento fatorial é dado por \[\mathbb{E}\left[(X)_r\right]=\mathbb{E}\left[X(X-1)(X-2)\cdots (X-r+1)\right]=\Biggr.\frac{\partial^r}{\partial t^r}\mathcal{M}_X(t)\Biggr|_{t=1}.\]

Dica

Seja \(Y=a+bX\), onde \(X\) é uma variável aleatória, \(a\) e \(b\) constantes, então \(\mathcal{M}_Y(t)=t^a\mathcal{M}_X(t^b)\).

Exemplo 2.16 Seja \(X\) uma variável aleatória de uma população com distribuição Binomial com parâmetros \(n\) e \(p\), i.e., \(\mathbb{P}[X=x]=\binom{n}{x}p^x(1-p)^{n-x}\mathbb{I}_{\{0,1,2,\ldots,n\}}(x)\). Utilize a FGMF para calcular o \(r\)-ésimo momento fatorial da distribuição de \(X\).

Solução.

A FGMF é dada por

\[\begin{align*} \mathcal{M}_X(t)&=\mathbb{E}\left[t^{X}\right]=\sum_{j=0}^n t^j \binom{n}{j}p^j(1-p)^{n-j}=\sum_{j=0}^n \binom{n}{j}(tp)^j(1-p)^{n-j}=[(1-p)+tp]^n, \quad t\in\mathbb{R}. \end{align*}\]

Daí,

\[\begin{align*} \mathbb{E}\left[X\right]&=\frac{\partial}{\partial t}\mathcal{M}_X(t)\Biggr|_{t=1}=n[(1-p)+tp]^{n-1}p\Biggr|_{t=1}=np;\\ \\ \mathbb{E}\left[(X)_2\right]&=\frac{\partial^2}{\partial t^2}\mathcal{M}_X(t)\Biggr|_{t=1}=n(n-1)p^2[(1-p)+tp]^{n-2}\Biggr|_{t=1}=n(n-1)p^2;\\ \\ &\,\,\,\,\vdots \\ \\ \mathbb{E}\left[(X)_r\right]&=\frac{\partial^r}{\partial t^r}\mathcal{M}_X(t)\Biggr|_{t=1}=n(n-1)\cdots(n-r+1)p^r=\frac{n!}{(n-r)!}p^r=(n)_r\,p^r. \quad\checkmark \end{align*}\]

Seja \(X\) uma variável aleatória de uma população com distribuição Poisson com parâmetro \(\lambda\), i.e. \(\mathbb{P}[X=x]=\frac{e^{-\lambda}\lambda^x}{x!}\mathbb{I}_{\{0,1,2,\ldots\}}(x)\). Utilize a FGMF para calcular o \(r\)-ésimo momento fatorial da distribuição de \(X\).

Solução.

A FGMF é dada por

\[\begin{align*} \mathcal{M}_X(t)&=\mathbb{E}\left[t^{X}\right]=\sum_{j=0}^\infty t^j \frac{e^{-\lambda}\lambda^j}{\lambda!}=e^{-\lambda}\sum_{j=0}^\infty \frac{(t\lambda)^j}{\lambda!}=e^{-\lambda}e^{t\lambda}=e^{(t-1)\lambda}, \quad t\in\mathbb{R}. \end{align*}\]

Dessa forma,

\[\begin{align*} \mathbb{E}\left[X\right]&=e^{(t-1)\lambda}\,\lambda\Biggr|_{t=1}=\lambda;\\ \\ \mathbb{E}\left[(X)_2\right]&=e^{(t-1)\lambda}\,\lambda^2\Biggr|_{t=1}=\lambda^2;\\ \\ &\,\,\,\,\vdots \\ \\ \mathbb{E}\left[(X)_r\right]&=\frac{\partial^r}{\partial t^r}\mathcal{M}_X(t)\Biggr|_{t=1}=e^{(t-1)\lambda}\,\lambda^r\Biggr|_{t=1}=\lambda^r. \quad\checkmark \end{align*}\]

Exemplo 2.17 Seja \(X\) uma variável aleatória de uma população com distribuição Binomial negativa com parâmetros \(n\) e \(p\), i.e., \(\mathbb{P}[X=x]=\binom{x+n-1}{x}p^n(1-p)^{x}\mathbb{I}_{\{0,1,2,\ldots\}}(x)\). Calcule a FGMF da distribuição de \(X\).

Solução.

A FGMF é dada por

\[\begin{align*} \mathcal{M}_X(t)&=\mathbb{E}\left[t^{X}\right]=\sum_{j=0}^\infty t^j \binom{j+n-1}{j}p^n(1-p)^{j}=\sum_{j=0}^\infty \binom{j+n-1}{j}p^n[t(1-p)]^{j}\\ &=p^n\sum_{j=0}^\infty \binom{j+n-1}{j}[t(1-p)]^{j}=p^n[1-(1-p)t]^{-n}=\left[\frac{p}{1-(1-p)t}\right]^n, \quad |t|<\frac{1}{1-p}. \quad\checkmark \end{align*}\]

A última soma é resultado de aplicar os resultados da Seção A.2 do Apêndice A.

Observação

Se \(X\) é uma variável aleatória discreta com valores em \(\{0,1,2,\ldots\}\), então \(\mathcal{M}_X(t)\) é chamada de função geradora de probabilidades (FGP). Nesse caso, \(\mathcal{M}_X(t)\) existe para todo \(|t|\le1\).

A \(r\)-ésima probabilidade de massa de \(X\) calcula-se como \[\mathbb{P}\left[X=r\right]=\frac{1}{k!}\frac{\partial^r}{\partial t^r}\mathcal{M}_X(t)\Biggr|_{t=0}.\]

Exemplo 2.18 Considere o cenário do Exemplo 2.16. Utilize a FGP da distribuição de \(X\) para calcular a \(r\)-ésima probabilidade de massa.

Solução.

Pelo Exemplo 2.16, sabemos que a FGP é dada por \(\mathcal{M}_X(t)=[(1-p)+tp]^n\). Daí,

\[\begin{align*} \mathbb{P}\left[X=0\right]&=[(1-p)+tp]^n\Biggr|_{t=0}=(1-p)^n; \\ \\ \mathbb{P}\left[X=1\right]&=\frac{\partial}{\partial t}\mathcal{M}_X(t)\Biggr|_{t=0}=np(1-p)^{n-1}; \\ \\ &\,\,\,\,\vdots \\ \\ \mathbb{P}\left[X=r\right]&=\frac{1}{r!}\frac{\partial^r}{\partial t^r}\mathcal{M}_X(t)\Biggr|_{t=0}=\frac{n(n-1)\cdots(n-r+1)}{r!}p^r(1-p)^{n-r}\\ &=\frac{n!}{(n-r)!r!}p^r(1-p)^{n-r}=\binom{n}{r}p^r(1-p)^{n-r}. \quad\checkmark \end{align*}\]

2.2.2 Cumulantes

Definição 2.4 (Função geradora de cumulantes) Seja \(X\) uma variável aleatória de uma população com função de distribuição \(F_X\). A função geradora de cumulantes (FGC) de \(F_X\), define-se como

\[K_X(t)=\log M_X(t)=\sum_{j=1}^\infty\kappa_n\frac{t^j}{j!},\]

onde \(M_X(t)\) representa a FGM de \(F_X\), \(|t|<h\), \(h>0\). As constantes \(\kappa_1,\kappa_2,\kappa_3,\ldots\) são chamados de cumulantes da distribuição \(F_X\).

Dica

Os cumulantes são funções dos momentos e podem ser obtidos diferenciando a função \(K_X(t)\)

\[\kappa_j=K'_X(0)=\frac{\partial^j}{\partial t^j}K_X(t)\Biggr|_{t=0},\] para \(j=1,2,3,\ldots\).

Exemplo 2.19 Sejam \(X_1,X_2\) variáveis aleatórias independentes. Mostre que \[K_{X+Y}(t)=K_X(t)+K_Y(t).\]

Solução.

Aplicando a Definição 2.4, temos que

\[\begin{align*} K_{X+Y}(t)&=\log\mathbb{E}[e^{t(X+Y)}]=\log\mathbb{E}[e^{tX}e^{tY}] =\log\left(\mathbb{E}[e^{tX}]\mathbb{E}[e^{tY}]\right)\\ &=\log\mathbb{E}[e^{tX}]+\log\mathbb{E}[e^{tY}]=K_X(t)+K_Y(t). \quad\checkmark \end{align*}\]

Exemplo 2.20 Seja \(X\) uma variável aleatória de uma população com distribuição Bernoulli com parâmetro \(p\). Utilize a FGC para calcular o \(r\)-ésimo cumulante da distribuição de \(X\).

Solução.

Pela Definição 2.3, sabemos que \(\mathcal{M}_X(t)=M_X(\log t)\). Pelo Exemplo 2.16, sabemos que a FGMF de uma variável aleatória com distribuição binomial é dada por \(\mathcal{M}_X(t)=[(1-p)+tp]^n\), com \(t\in\mathbb{R}\). Assim, a FGM de uma variável aleatória com distribuição Bernoulli é dada por \(M_X(t)=(1-p)+e^tp\), com \(t\in\mathbb{R}\). Daí, \(K_{X}(t)=\log M_X(t)=\log[(1-p)+e^tp]\).

Dessa forma,

\[\begin{align*} K'_{X}(0)&=\kappa_1=\frac{e^tp}{(1-p)+e^tp}\Biggr|_{t=0}=p\\ K''_{X}(0)&=\kappa_2=e^tp\frac{(1-p)+e^tp-e^tp}{((1-p)+e^tp)^2}\Biggr|_{t=0}=p(1-p)\\ K'''_{X}(0)&=\kappa_3=\frac{[(1-p)+e^tp]^2p(1-p)e^t-2p(1-p)e^t(e^tp+p(1-p)e^t)}{((1-p)+e^tp)^4}\Biggr|_{t=0}=p(1-p)(1-2p)\\ &\,\,\,\,\vdots \\ \\ K^{(r)}_X(0)&=\kappa_r=\frac{\partial^r}{\partial t^r}K_X(t)\Biggr|_{t=0}. \end{align*}\]

Para calcular a \(r\)-ésima derivada de \(K_X(t)\), note que

\[\begin{align*} \frac{\partial\kappa_r}{\partial p}&=\frac{\partial}{\partial p}\left[\frac{\partial^r}{\partial t^r}\log[(1-p)+e^tp]\Biggr|_{t=0}\right] =\frac{\partial^r}{\partial t^r}\left[\frac{\partial}{\partial p}\log[(1-p)+e^tp]\Biggr|_{t=0}\right]\\ &=\frac{\partial^r}{\partial t^r}\left[\frac{e^t-1}{(1-p)+e^tp}\Biggr|_{t=0}\right]. \quad\checkmark \end{align*}\]

Por outra parte,

\[\begin{align*} \kappa_{r+1}&=\frac{\partial^{r+1}}{\partial t^{n+1}}K_X(t)\Biggr|_{t=0} =\frac{\partial^r}{\partial t^r}\left[\frac{\partial}{\partial t}K_X(t)\Biggr|_{t=0}\right]\\ &=\frac{\partial^r}{\partial t^r}\left[\frac{e^tp}{(1-p)+e^tp}\Biggr|_{t=0}\right]. \quad\checkmark \end{align*}\]

Daí,

\[\begin{align*} \kappa_{r+1}-p(1-p)\frac{\partial\kappa_r}{\partial p}&=\frac{\partial^r}{\partial t^r}\left[\frac{e^tp}{(1-p)+e^tp}-p(1-p)\frac{e^t-1}{(1-p)+e^tp}\Biggr|_{t=0}\right]\\ &=\frac{\partial^r}{\partial t^r}\left[\frac{e^tp^2+p(1-p)}{(1-p)+e^tp}\Biggr|_{t=0}\right]=\frac{\partial^r}{\partial t^r}p=0. \quad\checkmark \end{align*}\]

Dessa forma, para \(r>1\) \[\kappa_{r+1}=p(1-p)\frac{\partial}{\partial p}\kappa_r. \quad\checkmark\]

Observação

Da Definição 2.2, sabemos que \(M_X(t)=\mathbb{E}\left[e^{tX}\right]\) existe para todo \(|t|<h\), para algum \(h>0\). Se \(\mathbb{E}X^r\) existe para todo \(r=1,2,\ldots\), então

\[M_X(t)=\sum_{r=0}^\infty \frac{t^r}{r!}\mathbb{E}X^r, \quad |t|<h.\] Dessa forma,

\[\begin{align*} K_X(t)&=\log M_X(t)=\log\left[\sum_{r=0}^\infty \frac{t^r}{r!}\mathbb{E}X^r\right] =\log\left[1+\sum_{r=1}^\infty \frac{t^r}{r!}\mathbb{E}X^r\right]\\ &=\mathbb{E}[X]\, t+\frac{1}{2!}\left[\mathbb{E}X^2-(\mathbb{E}X)^2\right]t^2 +\frac{1}{3!}\left[\mathbb{E}X^3-\mathbb{E}X\mathbb{E}X^2-2\left(\mathbb{E}X^2-(\mathbb{E}X)^2\right)\mathbb{E}X\right]t^3+\cdots. \end{align*}\]

Daí, \(\kappa_1=\mathbb{E}X\), \(\kappa_2=\mathbb{E}X^2-(\mathbb{E}X)^2\), \(\kappa_3=\mathbb{E}X^3-\mathbb{E}X\mathbb{E}X^2-2\left(\mathbb{E}X^2-(\mathbb{E}X)^2\right)\mathbb{E}X=\mathbb{E}\left[(X-\mathbb{E}X)^3\right]\), \(\kappa_4=\mathbb{E}\left[(X-\mathbb{E}X)^4\right]-3\kappa_2^2\), etc.

2.3 Função geradora de momentos conjunta

Definição 2.5 Seja \(\mathbf{X}=(X_1,X_2,\ldots,X_n)^\intercal\) uma vetor aleatório de tamanho \(n\times1\). A função geradora de momentos conjunta (FGMC) define-se, como

\[M_{\mathbf{X}}(\mathbf{t})=\mathbb{E}\left[e^{\mathbf{t}^{\intercal}\mathbf{X}}\right]=\mathbb{E}\left[e^{\sum_{i=1}^nt_iX_i}\right],\] se existe para todos os vetores reais \(\mathbf{t}=(t_1,t_2,\ldots,t_n)\) que pertencem a um retângulo rechado \(H\), tal que

\[H=[-h_1,h_1]\times[-h_2,h_2]\times\cdots\times[-h_n,h_n]\subset\mathbb{R}^n,\] com \(h_i>0\) para todo \(i=1,2,3,\ldots, n\).

Exemplo 2.21 Seja \(\mathbf{X}=(X_1,X_2)\) um vetor aleatório de uma população com função de densidade conjunta dada por

\[f(x_1,x_2)=e^{-(x_1+x_2)}, \quad x_1>0, x_2>0.\]

Calcule a FGMC da distribuição de \(\mathbf{X}=(X_1,X_2)\).

Solução.

Pela Definição 2.5, sabemos que

\[\begin{align*} M_{\mathbf{X}}(\mathbf{t}) &= \mathbb{E}\left[e^{\mathbf{t}^{\intercal}\mathbf{X}}\right] = \mathbb{E}[e^{t_1X_1+t_2X_2}] = \int_{0}^\infty \int_{0}^\infty e^{t_1x_1+t_2x_2}e^{-(x_1+x_2)}\,dx_1dx_2 \\ &= \int_{0}^\infty \int_{0}^\infty e^{(t_1-1)x_1}e^{(t_2-1)x_2}\,dx_1dx_2 = \int_{0}^\infty e^{(t_1-1)x_1}\,dx_1 \int_{0}^\infty e^{(t_2-1)x_2}\,dx_2\\ &= \frac{1}{(1-t_1)(1-t_2)}. \quad\checkmark \end{align*}\]

Exemplo 2.22 Seja \(X_1,X_2\) uma amostra aleatória de uma população com distribuição normal padrão. Encontre as distribuições de \(2X_1X_2\) e \(X_2^2-X_1^2\).

Solução.

Sabemos que \(X_1\) e \(X_2\) são variáveis aleatórias independentes e identicamente distribuidas com distribuição \(N(0,1)\). Dessa forma, \(2X_1X_2\) tem distribuição \(2\chi_{(1)}^2\), ou seja \(Gama\left(\frac12,\frac14\right)\).

Seja \(U=2X_1X_2\), então

\[\begin{align*} \mathbb{E}\left[e^{tU}\right]&=\mathbb{E}\left[e^{2tX_1X_2}\right]= \frac{1}{2\pi}\int_{-\infty}^\infty\int_{-\infty}^\infty e^{2tx_1x_2}e^{-\frac12(x_1^2+x_2^2)}\,dx_1dx_2\\ &=\frac{1}{2\pi}\int_{-\infty}^\infty e^{-\frac12x_2^2}\int_{-\infty}^\infty e^{-\frac12x_1^2}e^{2tx_1x_2}\,dx_1dx_2\\ &=\frac{1}{2\pi}\int_{-\infty}^\infty \exp\left\{-\frac12x_2^2\right\}\exp\left\{2t^2x_2^2\right\}\int_{-\infty}^\infty \exp\left\{-\frac12\left(x_1-2tx_2\right)^2\right\}\,dx_1dx_2\\ &=\frac{1}{\sqrt{2\pi}}\int_{-\infty}^\infty \exp\left\{-\frac12\left(1-4t^2\right)x_ 2^2\right\}\,dx_2\\ &=\left(1-4t^2\right)^{-\frac12}, \quad t^2<\frac14 \text{ ou } |t|<\frac12. \quad\checkmark \end{align*}\]

Seja \(t^*=t^2\), então \(\mathbb{E}\left[e^{t^*U}\right]=\left(1-4t^*\right)^{-\frac12}\), com \(t^*<\frac{1}{4}\), i.e., a função geradora de momentos de uma \(Gama(\frac12,\frac14)\).

Considere \(V=X_2^2-X_1^2\), então

\[\begin{align*} \mathbb{E}\left[e^{tV}\right]&=\mathbb{E}\left[e^{t\left(X_2^2-X_1^2\right)}\right]=\mathbb{E}\left[e^{tX_2^2}\right]\mathbb{E}\left[e^{- tX_1^2}\right]=\left(1-2t\right)^{-\frac12}\left(1+2t\right)^{-\frac12}\\ &=\left[\left(1-2t\right)\left(1+2t\right)\right]^{-\frac12}=\left(1-4t^2\right)^{-\frac12}, \quad t^2<\frac14 \text{ ou } |t|<\frac12. \end{align*}\]

Pelo @-fgm_unicidade, temos que \(U\) e \(V\) seguem a mesma distribuição \(2\chi_{(1)}^2\) ou \(gama(\frac12,\frac14)\quad\checkmark\).

Exemplo 2.23 Seja \(X_1,X_2\) uma amostra aleatória de uma população com distribuição normal padrão. Calcule a FGMC do vetor \((Y,Z)\), onde \(Y=X_1+X_2\) e \(Z=X_1^2+X_2^2\). As variáveis \(Y\) e \(Z\) são correlacionadas?

Solução.

Sabemos que,

\[\begin{align*} M(t_1,t_2)&=\mathbb{E}\left[e^{t_1Y+t_2Z}\right]=\mathbb{E}\left[e^{t_1(X_1+X_2)+t_2\left(X_1^2+X_2^2\right)}\right]\\ &=\mathbb{E}\left[\exp\left\{t_1X_1+t_1X_2+t_2X_1^2+t_2X_2^2\right\}\right]\\ &=\mathbb{E}\left[\exp\left\{t_1X_1+t_2X_1^2\right\}\right]\mathbb{E}\left[\exp\left\{t_1X_2+t_2X_2^2\right\}\right]. \end{align*}\]

Por outra parte, temos que

\[\begin{align*} \mathbb{E}\left[\exp\left\{t_1X_1+t_2X_1^2\right\}\right]&=\int_{-\infty}^\infty e^{t_1x_1+t_2x_1^2}\frac{1}{\sqrt{2\pi}} e^{-\frac12x_1^2}\,dx_1\\ &=\frac{1}{\sqrt{2\pi}}\int_{-\infty}^\infty \exp\left\{t_1x_1+\left(t_2-\frac12\right)x_1^2\right\}\,dx_1\\ &=\frac{1}{\sqrt{2\pi}}\int_{-\infty}^\infty \exp\left\{-\frac12(1-2t_2)\left(x_1^2-\frac{2t_1}{1-2t_2}x_1\right)\right\}\,dx_1, \quad t_1\in\mathbb{R}, t_2\in\left(-\infty,\frac12\right)\\ &=\frac{1}{\sqrt{2\pi}}\int_{-\infty}^\infty \exp\left\{-\frac12(1-2t_2)\left[\left(x_1-\frac{t_1}{1-2t_2}\right)^2-\frac{t_1^2}{(1-2t_2)^2}\right]\right\}\,dx_1\\ &=\frac{1}{\sqrt{1-2t_2}}\exp\left\{\frac12\frac{t_1^2}{1-2t_2}\right\}. \quad \checkmark \end{align*}\]

Daí,

\[\begin{align*} M(t_1,t_2)&=\mathbb{E}\left[\exp\left\{t_1X_1+t_2X_1^2\right\}\right]\mathbb{E}\left[\exp\left\{t_1X_2+t_2X_2^2\right\}\right]\\ &=\left(\frac{1}{\sqrt{1-2t_2}}\exp\left\{\frac12\frac{t_1^2}{1-2t_2}\right\}\right)^2\\ &=\frac{1}{1-2t_2}\exp\left\{\frac{t_1^2}{1-2t_2}\right\}, \quad t_1\in\mathbb{R}, t_2\in\left(-\infty,\frac12\right). \quad\checkmark \end{align*}\]

Observe que, \(M(t_1,0)=M(t_1)=e^{t_1^2}\), para \(t_1\in\mathbb{R}\), i.e. \(Y\) é uma variável aleatória de uma população com distribuição normal com média \(0\) e variância \(2\). Por otro lado, \(M(,t_2)=M(t_2)=\frac{1}{1-2t_2}\), para \(-\infty<t_2<\frac12\), i.e., \(Z\) é uma variável aleatória de uma população com distribuição qui-quadrado com \(2\) graus de liberdade.

Para verificar se \(Y\) e \(Z\) são correlacionadas, calcula-se a correlação entre as duas variáveis aleatórias. Assim,

\[\begin{align*} \mathbb{E}YZ=\frac{\partial^2}{\partial t_1\partial t_2}M(t_1,t_2)\Biggr|_{(t_1,t_2)=(0,0)} &=\frac{\partial}{\partial t_1}\left[\frac{\partial}{\partial t_2}M(t_1,t_2)\right]\Biggr|_{(t_1,t_2)=(0,0)}\\ &=\frac{4t_1\exp\left\{\frac{t^2_1}{1-2t_2}\right\}(-4t_2+t_1^2+2)}{(1-2t_2)^4}\Biggr|_{(t_1,t_2)=(0,0)}\\ &=0. \quad\checkmark \end{align*}\]

Dessa forma, \(\mathrm{Cov}[Y,Z]=\mathbb{E}YZ-\mathbb{E}Y\mathbb{E}Z=0\). Daí, as variáveis \(Y\) e \(Z\) são não-correlacionadas.
